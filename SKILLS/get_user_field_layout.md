# get_user_field_layout

Retrieve the single global field layout used by Craft users.

## Endpoint

`get_user_field_layout`

## Parameters

None.

## Returns

Returns the global user field layout tabs and elements, plus the control-panel settings URL.

## Notes

- Users share one global field layout.
- The returned `fieldLayout.id` is a stable placeholder identifier for field-layout mutation tools.
